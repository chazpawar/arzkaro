# Source Code

This directory contains all the frontend source code for the Arzkaro app.

## Structure

```
src/
├── components/        # Reusable UI components
│   ├── Button.tsx
│   └── Card.tsx
├── constants/         # App constants (colors, styles, etc.)
│   ├── Colors.ts
│   └── Styles.ts
├── hooks/            # Custom React hooks
├── services/         # API services and integrations
├── utils/            # Utility functions
└── types/            # TypeScript type definitions
```

## Components

Reusable UI components following the design system.

## Constants

Design system tokens including colors, typography, spacing, and shadows.

## Hooks

Custom React hooks for shared logic across components.

## Services

API integrations, Supabase client, and other external services.

## Utils

Helper functions and utilities used throughout the app.

## Types

TypeScript type definitions and interfaces.
